SK海力士赴美发 ADR，把 AI 存储超级周期搬上华尔街

发生了什么

韩国存储大厂 SK 海力士周一宣布，将在美国发行近 1780 万股 ADR，每份相当于 1/10 普通股，预计周四定价、周五开始交易；按上周五首尔收盘价估算，可能募资约 280 亿美元。公司一季度营收同比增长近 200%，年内股价上涨约 260%。

AI为什么重要 · AI 分析

本质是把韩国 AI 存储周期的定价权部分交给美国资本市场，也让美股投资者多了一个直接押注 HBM 景气的标的，与已被爆炒的美光形成对照。风险同样明确：5500 亿美元产能要数年才落地，若届时 AI 内存需求结构生变，可能从供不应求反转为过剩、价格跳水。

Vercel CEO：每天一半部署由编码 Agent 触发，模型正与 Agent 解耦

发生了什么

云平台 Vercel 的 CEO 透露，公司目前每天处理约 600 万次部署，其中一半由编码 Agent 触发，AI 网关每天流经超过 1 万亿 token。他称去年是“到处做原型”，今年重点转向让 Agent 在生产环境中稳定可用。

AI为什么重要 · AI 分析

一半部署由编码 Agent 发起、日均万亿 token，说明写代码的 Agent 已是当下 token 消耗的主引擎。他点出的“模型与 Agent 解耦”会削弱前沿实验室靠模型锁定客户的护城河，议价权向做编排和基础设施的中间层转移。
