英伟达发布新一代 AI 芯片 Rubin，训练吞吐较上代提升约 2.5 倍

事实黄仁勋在年度发布会上正式发布代号 Rubin 的数据中心 GPU，官方称训练吞吐较上代 Blackwell 提升约 2.5 倍，采用 HBM4 显存、以整机柜方案 NVL 系列交付，并计划于明年规模量产。

展开精读

Rubin 采用全新架构，搭配带宽更高的 HBM4 显存，单卡显存容量与互连带宽较上代 Blackwell 均有提升。整机柜方案 NVL 系列同步更新，配套交换芯片、网络与液冷一并发布，英伟达强调本代"以机柜为单位"交付整套算力，而非单卖加速卡。

按官方给出的路线图，Rubin 计划于明年进入规模量产，并与现有 Blackwell 平台并行供货一段时间；同场还公布了配套的 CPU、网络与软件栈更新，延续"每年一代"的迭代节奏。会上未披露具体定价，仅表示将随量产爬坡逐步放量。

“我们这一代的交付单位是整个机柜，而不是单颗芯片。”—— 黄仁勋，发布会主题演讲

会后多家头部云厂商表示将率先部署 Rubin。英伟达称，量产爬坡的进度主要取决于台积电先进封装（CoWoS）与 HBM4 显存的供给节奏；上游产能而非终端需求，是本轮交付的关键约束。

影响 · AI 分析（非原文事实）若量产顺利，将进一步巩固英伟达在 AI 训练市场的主导地位，主要云厂商资本开支预期或继续上修；对台积电先进封装产能是直接利好，竞品追赶窗口进一步收窄。

半导体 · 产能　／ 02

台积电上调全年资本开支，先进封装产能再度扩张

事实台积电将全年资本开支区间上调，增量主要投向先进封装与海外新厂，以应对 AI 芯片需求。

展开精读

台积电在业绩说明会上将全年资本开支区间上调，管理层表示先进封装 CoWoS 产能年内已翻倍仍供不应求，订单能见度延伸至明年；新增投资主要投向先进封装与海外新厂。

制程方面，N2（2nm）进展符合预期、明年放量，N3 需求维持高位。海外建厂的成本与人力压力被反复提及，但管理层重申不改变整体扩张节奏。

影响 · AI 分析（非原文事实）与英伟达量产计划相互印证，AI 算力链上游仍是卖方市场；对英伟达供给确定性是正面信号。

大模型 · 商业　／ 03

OpenAI 发布新模型并下调 API 价格，开发者生态再起波澜

事实OpenAI 推出新一代模型，宣称推理能力提升的同时显著下调 API 价格，并放开部分企业级功能。

展开精读

OpenAI 推出新一代模型，官方称在推理、代码与多模态等多项基准上刷新成绩，并同步下调主力模型的 API 调用价格，部分档位降幅过半；企业级的微调、长上下文等功能一并放开。

公司表示新模型已向付费开发者全量开放，免费层逐步推送。围绕训练数据来源与安全评估的监管讨论仍在持续，多地监管机构要求补充合规材料。

影响 · AI 分析（非原文事实）价格战或加速大模型应用普及，但也压缩中小模型厂商生存空间；与监管拉锯可能影响落地节奏。
